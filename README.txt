This is a front-end demo for DeForce featuring a fitness registration form.

Content:

- 2 HTML5 templates - for the form itself (index.html) and success page (success.html)
- style.css - for custom CSS and Google Font import
- script.js - JS file, mainly for the form steps propagation

Tech stack:

- HTML5
- CSS 3 + Bootstrap 5.3 from CDN
- jQuery 3.7 from CDN
- FontAwesome 6 from CDN
- Google Fonts from CDN repo
- animated SVG icon
